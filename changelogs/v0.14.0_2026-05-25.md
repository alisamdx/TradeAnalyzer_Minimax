# v0.14.0 — 2026-05-25

## LEAPS + CSP Strategy Screener

Implements the full LEAPS + CSP Strategy module as specified in `Requirements/SRS LEAPS CSP Strategy v2.docx`.

### Philosophy

This is an **opportunity ranker**, not a capital manager. The screener surfaces all qualifying LEAPS + CSP pairs ranked by combined score. The user decides how much capital to deploy. CSP leg can be on a **different ticker** than the LEAPS leg (cross-ticker pairing).

### Market Gate

Evaluates market conditions before screening:

| Signal | PASS | CAUTION | FAIL |
|--------|------|---------|------|
| SPY vs 50d MA | Above | — | Below |
| SPY vs 200d MA | Above | — | Below |
| VIX level | <20 | 20–30 | >30 |
| VIX 5d change | <+5 pts | — | >+5 pts |
| HYG/IEF ratio trend | Rising | Flat | Falling |

Gate result: **PASS** (all green), **FAIL** (any red), **CAUTION** (otherwise). Gate affects display only — all modes still run.

### LEAPS Screening

- Universe: tickers from most-recent screener run (`screen_results` JOIN `screen_runs`)
- Universe filters: price ≥ $10, ADV ≥ 2M shares, mkt cap ≥ $10B
- Hard-fail filters: delta 0.70–0.90, DTE 365–730, bid/ask spread ≤ 5%, OI ≥ 100, extrinsic ≤ 15%
- Expiry targets: 3rd Friday of January + June for next 3 years, filtered to 365–730 DTE window

### CSP Screening

- Same universe pool as LEAPS
- Hard-fail filters: delta −0.15 to −0.30, DTE 25–50, annualised return ≥ 12%
- Expiry: next Friday with DTE 25–50
- Pairing: same ticker as LEAPS (preferred) + up to 5 cross-ticker alternatives

### Scoring

**LEAPS sub-score** (weights sum to 100%):

| Component | Weight |
|-----------|--------|
| Trend strength (SMA stack) | 20% |
| Delta in target band | 15% |
| Extrinsic % (lower = better) | 20% |
| IV state (IVR) | 15% |
| Liquidity (OI + spread) | 10% |
| Fundamentals (ROE, D/E) | 10% |
| Earnings distance | 10% |

**CSP sub-score** (weights sum to 100%):

| Component | Weight |
|-----------|--------|
| Annualised return | 25% |
| IVR | 20% |
| Delta in target band | 15% |
| Liquidity | 10% |
| Stock trend | 10% |
| Strike quality (OTM%) | 10% |
| Events calendar | 10% |

**Combined score** = (LEAPS sub × 0.60) + (CSP sub × 0.40), clamped 0–10.

**Grades**: A+ ≥9.0, A ≥8.0, B ≥7.0, C ≥6.0, F <6.0.

### Caution Flags

- `HIGH_LEAPS_PREMIUM` — LEAPS debit > 85% of stock price
- `LEAPS_IV_ELEVATED` — LEAPS IVR > 70
- `CSP_STRIKE_NEAR_PRICE` — CSP strike within 3% of stock price

### Implementation Notes

- **No earnings calendar**: Polygon has no public earnings endpoint. Earnings distance component scores 8 (neutral) for all tickers.
- **IVR approximation**: `(currentIV − iv52WkLow) / (iv52WkHigh − iv52WkLow) × 100` using Polygon's `details` endpoint.
- **SPY used as SPX proxy** for market gate calculations.
- **Rate limited**: All Polygon calls go through the existing `TokenBucketRateLimiter` (100 rpm default).
