# v0.7.0 — 2026-05-07

## Phase 7: Screener Enhancements (Priority 5)

### Sortable Columns

All screener result columns are now sortable (ascending/descending):
- Ticker, Company, Sector
- Last Price, Day Change %
- P/E Ratio, EPS
- Market Cap
- Revenue Growth %, EPS Growth %
- Debt/Equity, ROE %, Profit Margin %
- Free Cash Flow, Current Ratio
- Average Volume, Beta
- Pass Score (soft mode only)

Click any column header to sort. Click again to toggle direction. Visual indicator (▲/▼) shows current sort.

**Implementation:** Uses existing `useSortable` hook with payload flattening for sorting.

### Quick Actions

Each results row now has action buttons:
- ⭐ **Add to Watchlist** - Opens dropdown to select target watchlist
- 📊 **Run Analysis** - Triggers navigation to Analysis view with this ticker

### CSV Export

New "Export CSV" button exports all filtered/sorted results (not just current page).
Includes all columns plus Pass Score when in soft mode.

### Pagination

Results are paginated at 50 items per page:
- "Page X of Y" indicator
- Prev/Next navigation buttons
- Disabled state at boundaries
- Full sort applies before pagination

### Cache Status Integration

- Cache status indicator added to Screener header
- Polls every 5 minutes for staleness
- Warning before running screen if cache > 1 hour old
- Suggests refresh if data is stale

### Files Changed

- `src/renderer/src/views/ScreenerView.tsx` - Major rewrite with sorting, pagination, quick actions
- `src/renderer/src/styles.css` - Added styles for sortable headers, pagination, quick actions, watchlist dropdown

### Dependencies

No new dependencies. Uses existing `useSortable` hook.

### UI/UX Improvements

- Sort indicators on column headers
- Sorted by label shown in toolbar
- Hover effects on sortable headers
- Quick action buttons with tooltips
- Dropdown menu for watchlist selection
- Pagination controls at bottom of results

### Notes

- Quick Actions "Run Analysis" emits a custom event that parent components can listen for
- CSV export uses Blob/URL API for download (no server required)
- Watchlist dropdown closes on selection or cancel
