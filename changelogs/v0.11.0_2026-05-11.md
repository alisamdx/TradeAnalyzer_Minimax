# v0.11.0 — 2026-05-11

## Feature: Local HTTP API Server

Adds a Fastify-based REST API server to the main process, enabling an external autonomous trading agent to drive the app programmatically without a UI.

### What was added

**`src/main/api-server.ts`** — `ApiServer` class:
- Binds exclusively to `127.0.0.1` (loopback only, never externally accessible).
- Authenticates every request via `X-Agent-Token` header (UUID written to `~/.tradeanalyzer/agent.token` on first launch).
- Writes the active port to `~/.tradeanalyzer/api.port` so the agent can discover it without hardcoding.
- Starts and stops with the Electron app lifecycle.
- Supports headless mode (`TRADEANALYZER_HEADLESS=1`) — main process + API server run without a BrowserWindow, for automated testing.

**`src/main/api/` — 10 route modules:**

| File | Endpoints |
|------|-----------|
| `routes-health.ts` | `GET /api/v1/health` — version, schema, market status, uptime |
| `routes-watchlists.ts` | `GET/POST /api/v1/watchlists`, `GET/POST/DELETE /api/v1/watchlists/:id/tickers` |
| `routes-screener.ts` | `POST /api/v1/screener/run`, `GET /api/v1/screener/presets`, `GET /api/v1/screener/presets/:id` |
| `routes-analysis.ts` | `POST /api/v1/analysis/run`, `GET/GET /api/v1/analysis/snapshots[/:id]` |
| `routes-validation.ts` | `GET /api/v1/validate/:ticker` — full 5-section validation dashboard as JSON |
| `routes-options.ts` | `GET /api/v1/options/:ticker` — filtered by DTE and delta query params |
| `routes-quotes.ts` | `POST /api/v1/quotes` — bulk quote fetch, cache-aware |
| `routes-fundamentals.ts` | `GET /api/v1/fundamentals/:ticker` — cached derived ratios |
| `routes-jobs.ts` | `POST/GET /api/v1/jobs`, `GET /api/v1/jobs/:id`, `POST /api/v1/jobs/:id/stop`, `POST /api/v1/jobs/:id/resume` |
| `routes-settings.ts` | `GET /api/v1/settings` — non-sensitive settings (API key is never returned) |

**`src/main/api/helpers.ts`** — shared `apiOk` / `apiError` / `fromServiceError` response helpers enforcing the unified response contract: `{ ok: true, data }` / `{ ok: false, error, code }`.

### Response contract

All endpoints return:
- **Success:** `{ "ok": true, "data": <payload> }`
- **Error:** `{ "ok": false, "error": "human-readable message", "code": "SNAKE_CASE_CODE" }`

### Security

- Loopback-only binding (`127.0.0.1`).
- All requests require `X-Agent-Token` header matching the UUID in `~/.tradeanalyzer/agent.token`. Missing or mismatched token returns HTTP 401.
- The Polygon API key is never returned by any endpoint.

### Dependencies

- **`fastify@^5.8.5`** added to `dependencies`.

### Port configuration

Default port `7432`. Overridable via the `apiServerPort` setting in the app database. Active port written to `~/.tradeanalyzer/api.port` on startup.
