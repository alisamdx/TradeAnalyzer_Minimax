# v0.20.0 — 2026-05-31

## Strategy Lab

### Summary

New **🔬 Strategy Lab** view in the Strategy sidebar section (just above 📚 Knowledge). Given any single ticker it fetches entirely fresh live data and scores all 31 tastylive option strategies algorithmically. Two modes: **Validate** (rank all 31) and **Explore** (build one concrete setup).

### Changes

**`src/shared/types.ts`**
- Added `StrategyLabDirectionBias`, `StrategyLabGrade`, `StrategyLabComplexity`
- Added `StrategyLabContext`, `SetupLeg`, `StrategySetup`, `StrategyScore`, `StrategyLabValidateResult`

**`src/main/services/strategy-lab-service.ts`** *(new)*
- `StrategyLabService` class — `validate(db, ticker)`, `explore(db, ticker, slug)`, `getContext(db, ticker)`
- 31 `StrategyProfile` definitions covering bullish, bearish, neutral, neutral-bullish, neutral-bearish, omnidirectional categories
- Helpers: `selectExpiration` (nearest 20–50 DTE, fallback ≥14 DTE), `computeDirectionBias` (MA20/MA50 + slope), `computeAtmIv` (ATM chain IV), `computeIvRank` (from `iv_history` last 252 rows), `findByDelta`, `buildSetup` (full leg construction + P&L), `scoreStrategy` (IV 30pt + direction 30pt + premium 25pt + liquidity 15pt)
- Data sources: fresh `getHistoricalBars(90d)` + live `getOptionsExpirations` + `getOptionsChain` + `iv_history` table — zero cache reliance

**`src/main/ipc/ipc-strategy-lab.ts`** *(new)*
- `strategyLab:validate(ticker)` — score all 31 strategies
- `strategyLab:explore(ticker, slug)` — single-strategy concrete setup
- `strategyLab:aiRationale(score, ctx)` — Claude `claude-haiku-4-5` call, ~$0.003–0.005/click, requires Anthropic API key in Settings

**`src/main/index.ts`**
- Import + `registerStrategyLabIpc(db, dataProvider, optionsProvider)`

**`src/preload/index.ts`**
- Added `strategyLab` bridge: `validate`, `explore`, `aiRationale`
- Imported Strategy Lab types

**`src/renderer/src/views/StrategyLabView.tsx`** *(new)*
- **Validate tab**: ticker input → Run Validate → `ContextCard` (ticker, price, expiry, ATM IV, IV rank, direction bias) + scored strategy cards with grade badge, score bars (IV/Direction/Premium/Liquidity), condition flag chips, expandable leg table, `→ Explore Strategy` / `📈 View in Payoff` / `✨ AI Rationale` buttons
- **Explore tab**: ticker + strategy dropdown (all 31) → `Build Setup` → legs table + P&L summary + same action buttons; cross-linked from Validate "→ Explore" with slug pre-selected
- `navigate-to-payoff` custom DOM event reused for Payoff navigation

**`src/renderer/src/App.tsx`**
- Added `'strategyLab'` to `View` type union
- Imported `StrategyLabView`
- Sidebar: `🔬 Strategy Lab` button above `📚 Knowledge` in Strategy section
- Router: `case 'strategyLab': return <StrategyLabView />`
