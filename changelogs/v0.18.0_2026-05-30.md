# v0.18.0 — 2026-05-30

## ENH-2: Opportunity Dashboard

### Summary
Replaced the Morning Briefing screen with a live Opportunity Dashboard that ranks every universe ticker by a composite trade-quality score. The entire scoring pipeline runs against the local SQLite DB — no additional API calls are made.

### New Files
- `src/main/services/opportunity-service.ts` — Composite scoring engine. Batch-queries iv_history (window function for 252-row per-ticker IV rank), quote_cache (prices), screen_results (fundamentals), and analysis_snapshots (technical scores). Computes weighted composite: fundamentals 25% + IV rank 30% + technical 25% + premium yield 20%.
- `src/main/ipc/ipc-opportunity.ts` — IPC handler for `opportunity:run`.
- `src/renderer/src/views/OpportunityView.tsx` — Ranked table with composite score circles, mini score bars, IV rank badges, estimated premium/strike, one-click drill-in to Analysis and Options Chain views.

### Modified Files
- `src/shared/types.ts` — Added `OpportunityRow`, `OpportunityRunOptions`, `StrategyMode`, `OpportunityUniverse` types.
- `src/preload/index.ts` — Added `window.api.opportunity.run()`. Removed `window.api.briefing.*`.
- `src/main/index.ts` — Instantiates `OpportunityService`, calls `registerOpportunityIpc`. Removed `registerBriefingIpc`.
- `src/renderer/src/App.tsx` — Replaced `BriefingView` with `OpportunityView`. Sidebar button: 📰 Briefing → 🎯 Opportunity.

### Removed Files
- `src/renderer/src/views/BriefingView.tsx` — Deleted.

### Scoring Details
| Component | Weight | Source |
|-----------|--------|--------|
| Fundamentals | 25% | Latest screen_results: ROE, P/E, D/E, profit margin, revenue growth |
| IV Rank | 30% | iv_history (last 252 rows): IVR directly for premium selling; 100-IVR for directional |
| Technical | 25% | Latest analysis_snapshots: compositeScore (buy mode) or suitabilityScore (wheel mode) |
| Premium Yield | 20% | Estimated: 1.5% of 92%-strike per month, annualized, scaled vs 3%/mo target |

### Strategy Modes
- **Wheel / CSP / Spreads**: High IV rank = good (sell rich premium)
- **Bullish / Bearish**: Low IV rank = good (buy cheap options)

### Controls
- Universe: S&P 500 / Russell 1000 / Both
- Strategy: Wheel / CSP / Spreads / Bullish / Bearish
- Min composite score filter
- Top N results (25 / 50 / 100 / 200)
