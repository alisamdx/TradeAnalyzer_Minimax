# v0.19.2 — 2026-05-31

## History Screen Overhaul + Bulk Price Load

### Summary

Renamed "IV History" → "History". Removed the gap-fill run button from the History screen (already present in Data Sync). Added bulk price history load (2Y, full universe) to History screen. Added price gap-fill section to Data Sync.

### Changes

**`src/renderer/src/views/IvHistoryView.tsx`**
- Exported function renamed `IvHistoryView` → `HistoryView` (backward-compat alias kept)
- Removed "Ongoing Refresh — Gap Fill" SectionCard (button already existed in DataSync section 3)
- Removed `gap_fill` from `Phase` union type and `PHASE_LABELS`
- Added labeled section dividers ("IV History" / "Price History") in left column
- Added `IV Coverage` card showing gap count and pointer to Data Sync for gap fill
- Added **Bulk Price Load** SectionCard: universe selector (S&P 500 / Russell 1000 / Both), fixed 2Y range, Start / Cancel button, coverage count (`historical:getPriceTickerCount`)
- Progress panel and console shared between IV phases and price bulk load
- New state: `bulkUniverse`, `bulkRunning`, `bulkProgress`, `bulkResult`, `priceCoverage`, `bulkCancelRef`
- `startBulkPrice()` — gets tickers via `historical:getUniverseTickers`, loops with `fetchPrices('2Y')`, cancelable via ref

**`src/renderer/src/views/DataView.tsx`**
- Added **section 5: Price Gap Fill** — finds stale tickers via `historical:getStalePriceTickers`, loops calling `fetchPrices('1M')`, shows stale count + progress bar + result summary
- Renumbered "Local Database Stats" from section 5 → **section 6**
- Added `loadPriceGapStatus()`, `runPriceGapFill()`, `cancelPriceGapFill()`
- New state: `priceStaleCount`, `priceGapRunning`, `priceGapProgress`, `priceGapResult`, `priceGapCancelRef`

**`src/renderer/src/App.tsx`**
- Sidebar label: `📊 IV History` → `📊 History`

**`src/main/ipc/ipc-historical.ts`**
- `historical:getUniverseTickers(universe)` — queries `constituents` table for all tickers in the given universe
- `historical:getStalePriceTickers()` — tickers in `historical_prices` where `MAX(date) < date('now', '-2 day')`
- `historical:getPriceTickerCount()` — count of distinct tickers in `historical_prices`

**`src/preload/index.ts`**
- Added `window.api.historical.getUniverseTickers(universe)`
- Added `window.api.historical.getStalePriceTickers()`
- Added `window.api.historical.getPriceTickerCount()`
