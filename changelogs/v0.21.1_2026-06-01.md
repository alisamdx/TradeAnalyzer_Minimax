# v0.21.1 — 2026-06-01

## fix: Batch Jobs — Run History above Live Activity

Reordered the panels in `BatchView` so Run History appears between the job table and the Live Activity log. Previously Run History was at the bottom, hidden below the (conditionally rendered) Live Activity panel whenever a job was running or had just finished. Now the historical record is always visible without scrolling.

### Changed
- `src/renderer/src/views/BatchView.tsx` — panel order: Registered Jobs → Run History → Live Activity
