# v0.16.0 — 2026-05-30

## Payoff Visualizer — Fixes & Assessment Panel Redesign

- Fixed Anthropic API key injection: `new Anthropic({ apiKey })` — previously relied on env var which may not be set in the Electron context
- Fixed `probOfProfit` tool schema: changed from `type: ['string', 'null']` to `type: 'string'` with empty-string fallback (Claude tool use does not accept union types)
- Redesigned assessment panel: structured cards for Opportunities / Key Risks / Exit Guidance / Ideal Market Conditions
- Collapsible options chain drawer (expanded by default, user can collapse)
- SVG payoff chart layout constants extracted for maintainability

## E*Trade Quote Inspector (Test API Screen)

- New IPC handler `etrade:get-raw-quote`: calls `/v1/market/quote/{symbol}?detailFlag=ALL` via the existing OAuth 1.0a signing infrastructure and returns the full raw JSON response
- New `window.api.etrade.getRawQuote(symbol)` preload bridge
- New `QuoteInspector` component in `TestApiView` (E*Trade tab, shown when authenticated):
  - Enter a ticker → Fetch Quote → see every field E*Trade returns
  - Automatically highlights any IV / volatility-related fields found anywhere in the nested response
  - Full collapsible raw JSON viewer
- **Finding:** E*Trade `detailFlag=ALL` returns price, fundamental, and extended-hours data only — no IV rank, IV percentile, or volatility history. Confirmed E*Trade API cannot serve as an IV rank source.
