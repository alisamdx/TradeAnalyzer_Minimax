# v0.19.1 — 2026-05-31

## Closed Positions Sync from E*Trade

Adds the ability to pull YTD closed trades from E*Trade transaction history and reconstruct them as closed positions in the local portfolio DB.

### What changed

**`src/main/services/etrade-portfolio-service.ts`**
- `syncClosedPositions(db, creds, accountIdKey?)` — fetches full YTD transaction history, classifies each tx as open/close/expired/assigned, pairs legs FIFO per OCC symbol, inserts matched closed positions
- `fetchTransactionLegs()` — calls `/v1/accounts/{id}/transactions` with date range (Jan 1 YTD → today), pagination via `marker` field, max 50 tx per page
- `classifyTransaction()` — parses OCC symbol (`AAPL250620P00150000`), determines action type (open/close/expired/assigned)
- `matchClosedPositions()` — FIFO pairing of open+close legs per symbol
- `insertClosedPosition()` — dedup check on `ticker + position_type + entry_date + exit_date` before INSERT with `status = 'closed'`

**`src/main/ipc/ipc-portfolio.ts`**
- Added `portfolio:etrade:sync-closed` IPC handler

**`src/preload/index.ts`**
- Added `window.api.portfolio.etrade.syncClosed(accountIdKey?)` bridge

**`src/renderer/src/views/PortfolioView.tsx`**
- Added `syncClosedBusy` state
- Added `handleSyncClosed()` async handler
- Added **↓ Import Closed (YTD)** button on Closed tab
- Updated empty-state message for closed tab
