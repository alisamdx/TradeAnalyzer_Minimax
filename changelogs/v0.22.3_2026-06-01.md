# v0.22.3 — 2026-06-01

## feat: Single instance lock

Launching a second instance of TradeAnalyzer (e.g. clicking the shortcut while the app is already open) now focuses the existing window instead of opening a duplicate.

### Implementation
- `src/main/index.ts` — calls `app.requestSingleInstanceLock()` before `whenReady`.
  - If the lock is NOT acquired (another instance is already running): `app.quit()` immediately.
  - If the lock IS acquired: listens for `second-instance` event and brings the existing window to the front (`restore()` if minimized, then `focus()`).
