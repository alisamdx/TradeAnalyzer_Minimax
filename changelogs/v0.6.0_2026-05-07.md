# v0.6.0 — 2026-05-07

## Phase 6: Historical Charts (Priority 4)

### Database Migration

**`005_historical_data.sql`**
- `historical_financials` table: stores quarterly/annual financial data from Polygon
  - Revenue, Net Income, Gross Profit, Operating Income
  - EPS, Shares Outstanding
  - Total Assets, Liabilities, Shareholders Equity, Long-term Debt
  - Current Assets/Liabilities
  - Operating Cash Flow, Free Cash Flow, EBITDA
  - Metadata: currency, source, fetched_at, created_at
- `historical_prices` table: daily OHLCV bars
  - Open, High, Low, Close, Volume
  - Adjusted close for total return calculations
- Indexes for efficient querying:
  - `idx_historical_financials_ticker`, `idx_historical_financials_period`
  - `idx_historical_prices_ticker`, `idx_historical_prices_date`
- Views:
  - `v_latest_financials` - most recent quarterly financials per ticker
  - `v_latest_price` - latest price per ticker

### Backend Services

**`HistoricalDataService`** (`src/main/services/historical-service.ts`)
- `getFinancials(params)` - retrieve financial data for a ticker
- `upsertFinancial(financial)` - insert/update financial record
- `getLatestFinancialDate(ticker, periodType)` - check freshness
- `getPrices(params)` - retrieve OHLCV bars for date range
- `upsertPrice(price)` - insert/update price record
- `getLatestPriceDate(ticker)` - check freshness
- `calculateSMA(prices, period)` - calculate Simple Moving Average
- `getDateRangeFromTimeRange(range)` - convert 1M/3M/6M/1Y/2Y/5Y to dates
- `needsRefresh(latestDate, maxAgeDays)` - determine if data is stale

**`fetchAndStoreFinancials`** - Fetches from Polygon `/vX/reference/financials` API
- Maps Polygon financial structure to database schema
- Extracts income statement, balance sheet, cash flow data
- Calculates free cash flow from operating cash flow and capex

**`fetchAndStorePrices`** - Fetches from Polygon `/v2/aggs/ticker/{ticker}/range/1/day/{from}/{to}` API
- Retrieves daily OHLCV bars for specified time range
- Stores VWAP as proxy for adjusted close

**IPC Handlers** (`src/main/ipc/ipc-historical.ts`)
- `historical:getFinancials` - Get financials (quarterly/annual)
- `historical:getFinancialsLatestDate` - Check data freshness
- `historical:fetchFinancials` - Fetch and store from Polygon
- `historical:getPrices` - Get price bars for date range
- `historical:getPricesWithSMA` - Get prices with 50-day SMA overlay
- `historical:getPricesLatestDate` - Check data freshness
- `historical:fetchPrices` - Fetch and store from Polygon
- `historical:fetchAndStore` - Generic fetch/store for both types
- `historical:needsRefresh` - Check if data needs refresh

### Frontend Components

**`HistoricalFinancialChart`** (`src/renderer/src/components/HistoricalFinancialChart.tsx`)
- Recharts Area chart with gradient fill
- Metrics available:
  - Revenue
  - Net Income
  - Gross Profit
  - Operating Income
  - EPS (earnings per share)
  - Total Assets
  - Shareholders Equity
  - Free Cash Flow
  - EBITDA
- Period type selector: Quarterly / Annual
- Auto-refresh if data > 30 days old
- CSV export with date/value columns
- Large number formatting (K/M/B/T)

**`HistoricalPriceChart`** (`src/renderer/src/components/HistoricalPriceChart.tsx`)
- Recharts ComposedChart with price line and volume
- Time range selector: 1M, 3M, 6M, 1Y, 2Y, 5Y
- Features:
  - Price line (close prices)
  - Volume overlay (line with opacity)
  - 50-day SMA (dashed red line)
  - Current price and % change display
  - High/Low/Avg Volume stats
- Auto-refresh if data > 7 days old
- CSV export with OHLCV + SMA columns
- Custom tooltip with OHLCV data
