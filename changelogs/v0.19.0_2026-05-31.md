# v0.19.0 — 2026-05-31

## Knowledge Base

Embedded the tastylive Options Strategy Guide (TT1469) as an in-app quick-reference view.

### What was built

**Content extraction (offline, one-time):**
- 31 strategy pages extracted as 150 DPI PNGs → `src/renderer/src/knowledge/images/`
- 31 structured JSON one-pagers → `src/renderer/src/knowledge/data/`
- Both extracted from `docs/TT1469_strategy-guide_230627.pdf` pages 5–35

**Knowledge index (`src/renderer/src/knowledge/index.ts`):**
- `Strategy` interface with full typed schema (setup, risk/reward, Greeks, howItWorks, volatility, expiration, takeaways)
- `STRATEGIES` array (all 31), `STRATEGIES_BY_CATEGORY` grouped map, `CATEGORY_LABELS`/`CATEGORY_ORDER` constants

**`KnowledgeView.tsx`:**
- Collapsible left drawer (◀/▶ toggle) with search box and 6 color-coded category sections
- Active strategy highlighted with left border in category color
- Image / Text toggle in content header (color follows active category)
- **Image mode**: renders the original PDF page PNG (full visual fidelity)
- **Text mode**: structured layout — key params chips, setup steps, risk/reward 2×2 grid, Greeks 4-col row, How It Works cards, Volatility 2-col grid, At Expiration cards, Takeaways

**Sidebar reorganization (`App.tsx`):**
- Five labeled sections: Data / Analysis / Strategy / Settings / Personal
- `nav-section-label` CSS class: `—— LABEL ——` style dividers
- 📚 Knowledge added at bottom of Strategy section
